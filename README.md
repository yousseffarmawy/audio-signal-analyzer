# audio-signal-analyzer
Interactive audio signal analyzer using FFT, digital filtering, and time-frequency visualization.

## Overview
This project is an interactive desktop application for analyzing audio signals using FFT and digital signal processing techniques.

## Technologies
- Python
- NumPy
- SciPy
- PyQt5
- PyQtGraph

## Features
- Time domain visualization
- Frequency domain (FFT) analysis
- Digital filtering (Low-pass / High-pass)
- Spectrogram visualization
